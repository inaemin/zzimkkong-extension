# 찜꽁 회의실 레이더

찜꽁 예약 페이지에서 원하는 시간대의 회의실 상태를 빠르게 확인하고, 예약 입력을 더 쉽게 도와주는 Chrome 확장 프로그램입니다.

## 주요 기능

### 회의실 레이더

- 페이지 우측 하단에 회의실별 시간 블록 레이더를 표시합니다.
- 10분 단위로 예약 가능 / 예약 중 상태를 한눈에 볼 수 있습니다.
- 회의실 / 페어룸 탭 전환을 지원합니다.

### 빠른 시간 선택

- 타임블록을 클릭하면 1시간 단위로 날짜/시간/공간이 예약 폼에 바로 반영됩니다.
- 시간 범위를 바꾸려면 예약 폼에서 직접 수정합니다.

### Slack 메시지 복사

- 예약 생성/수정 성공 후 Slack 복사 모달을 자동으로 엽니다.
- 예약 수정 성공 시에는 guest booking 페이지로 복귀한 뒤 Slack 복사 모달을 한 번만 복구합니다.
- 예약 페이지에서는 `모달 테스트` 버튼으로 현재 입력값 기준 Slack 모달을 미리 열어볼 수 있습니다.
- `/remind` 명령 생성, 채널 선택, 알림 시점 선택을 지원합니다.
- 채널은 combobox 형태로 입력하며, 이전에 사용한 채널을 다시 선택하거나 새 `#채널`을 추가할 수 있습니다.
- 선택된 채널은 chip 형태로 표시되며, 저장된 채널은 dropdown에서 다시 선택하거나 삭제할 수 있습니다.
- 수정 후 guest 페이지가 blank 상태로 돌아오는 경우, 확장은 한 번 자동 새로고침해서 예약 페이지와 Slack 모달을 복구합니다.

### 회의실 메타데이터

- 회의실별 층 정보와 태그를 별도로 관리합니다.
- 현재 `창` 태그를 지원하며, 창문이 있는 회의실을 표시합니다.
- 13층 API 응답에 포함되지만 크루가 예약할 수 없는 방은 레이더에서 제외합니다.

현재 집계 대상 공간:

- 회의실: 금성, 지구, 수성, 화성, 보이저, 디스커버리, 아폴로, 허블, 은하수
- 페어룸: 페1 ~ 페14

13층 제외 공간: 목성, 스튜디오, 안드로메다같은방, 천왕성, 코치1, 코치2, 토성

## 사용 방법

1. 찜꽁 예약 페이지에 접속합니다.
2. 확장이 로드되면 우측 하단에 레이더가 자동으로 표시됩니다.
3. 날짜와 시간을 선택해 원하는 시간대의 회의실 상태를 확인합니다.
4. 필요하면 타임블록을 클릭해 예약 폼에 시간/공간을 반영합니다.
5. 예약 생성 또는 수정 완료 후 뜨는 Slack 모달에서 메시지를 복사해 사용합니다.

## 설치 방법

1. Chrome에서 `chrome://extensions` 열기
2. 우측 상단 `개발자 모드` 활성화
3. `압축해제된 확장 프로그램을 로드합니다` 클릭
4. 다운로드한 폴더 선택
5. 찜꽁 예약 페이지를 새로고침

## 참고 사항

- Slack 모달은 현재 채널 전용 UI입니다. 참석자 CSV import 기능은 더 이상 사용하지 않습니다.
- 예약 수정 후 guest 페이지가 드물게 blank 상태로 돌아오는 경우, 확장은 한 번 자동 새로고침해서 예약 페이지와 Slack 모달을 복구합니다.
- 찜꽁 서비스 업데이트 이후 일부 기능이 동작하지 않을 수 있습니다.

### GitHub 공개 전 확인

- `.env.example`을 복사해 `.env`를 만들고, 실제 guest URL/로컬 브라우저 경로 등 개인 설정은 `.env`에만 둡니다.
- `src-backup/`, `crews.csv`, `dist/`, `*.zip`, `test-results/`, `.DS_Store`, `.env*`는 공개 저장소에 올리지 않습니다.
- API 문서에는 실제 공유 맵 ID, 토큰, 비밀번호, 개인 식별자를 남기지 않고 placeholder를 사용합니다.
