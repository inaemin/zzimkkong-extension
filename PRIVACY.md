# 찜꽁 레이더 개인정보처리방침

시행일: 2026-07-07

찜꽁 레이더(이하 "확장 프로그램")는 찜꽁 예약 페이지에서 회의실/페어룸 예약 현황 확인, 예약 입력 보조, Slack 공유 메시지 작성을 돕기 위해 필요한 최소한의 정보만 처리합니다.

## 1. 수집·처리하는 정보

- 회의실 레이더 표시를 위해 찜꽁 예약 페이지와 찜꽁 API에서 제공되는 예약 관련 정보(날짜, 시간, 공간명, 예약 상태, 예약자명, 사용 목적 등)를 처리할 수 있습니다.
- 예약 폼 입력 보조를 위해 사용자가 예약 페이지에서 선택하거나 입력한 날짜, 시간, 공간 정보를 처리할 수 있습니다.
- 예약 생성/수정 후 Slack 공유 메시지를 만들기 위해 예약자명, 날짜, 시간, 공간명, 사용 목적 등 예약 페이지에 표시되는 정보를 처리할 수 있습니다.
- 사용 편의를 위해 최근 사용한 Slack 채널, 리마인더 설정, 레이더 UI 표시 상태, 마지막으로 선택한 회의실/페어룸 탭 같은 설정을 사용자의 브라우저 저장소에 저장할 수 있습니다.
- 사용자가 Slack 공유 메시지 복사 버튼을 클릭한 경우에만 생성된 공유 문구를 클립보드에 복사합니다.

## 2. 이용 목적

확장 프로그램은 위 정보를 다음 목적에만 사용합니다.

- 회의실/페어룸 예약 현황 표시
- 원하는 시간대의 회의실 확인 보조
- 예약 폼의 날짜, 시간, 공간 입력 보조
- 예약 완료 후 Slack 공유 메시지 생성
- 사용자가 선택한 UI 설정 및 최근 채널 유지

## 3. 보관 및 파기

- 예약 현황, 예약자명, 사용 목적 등 예약 페이지/API에서 읽은 정보는 기능 수행 중 브라우저 안에서 일시적으로 처리됩니다.
- 최근 Slack 채널, 리마인더 설정, 레이더 UI 표시 상태, 마지막 탭 선택값 등 편의 설정은 사용자의 브라우저 저장소에 보관될 수 있습니다.
- 브라우저 저장소에 보관된 설정은 사용자가 브라우저 데이터 삭제, 확장 프로그램 삭제, 사이트 데이터 삭제 등을 수행하면 삭제될 수 있습니다.
- 확장 프로그램은 개발자가 운영하는 별도 영구 저장소에 사용자 데이터를 저장하지 않습니다.

## 4. 제3자 제공 및 외부 전송

- 확장 프로그램은 개발자가 운영하는 별도 서버로 사용자 데이터를 전송하지 않습니다.
- 확장 프로그램은 사용자 데이터를 판매하지 않으며, 광고, 추적, 신용도 판단, 대출 심사 목적으로 사용하지 않습니다.
- 기능 수행에 필요한 경우 사용자가 접속한 찜꽁 서비스의 API(`k8s.zzimkkong.com`)로 예약 현황 조회 요청을 보낼 수 있습니다.
- Slack 공유 메시지는 사용자가 직접 복사해 사용하는 텍스트이며, 확장 프로그램이 Slack API를 통해 메시지를 자동 전송하지 않습니다.

## 5. 원격 코드 사용

확장 프로그램은 패키지에 포함된 JavaScript와 CSS 파일만 실행합니다. 외부 서버에서 JavaScript 또는 Wasm 파일을 불러와 실행하지 않으며, `eval()` 또는 문자열 기반 코드 실행을 사용하지 않습니다.

## 6. 이용자 권리

- 사용자는 언제든지 확장 프로그램을 비활성화하거나 삭제하여 정보 처리를 중단할 수 있습니다.
- 사용자는 브라우저의 사이트 데이터 또는 확장 프로그램 관련 데이터를 삭제하여 저장된 편의 설정을 삭제할 수 있습니다.
- Slack 공유 메시지 복사는 사용자가 복사 버튼을 클릭한 경우에만 실행됩니다.

## 7. 비공식 서비스 고지

확장 프로그램은 찜꽁 공식 제품이 아니며, 찜꽁의 제휴·승인·후원을 의미하지 않습니다. 찜꽁 서비스의 화면 구조나 동작 방식이 변경되면 일부 기능이 정상 동작하지 않을 수 있습니다.

## 8. 문의

문의 또는 개선 제안은 Chrome Web Store 게시자 정보 또는 프로젝트 저장소의 이슈를 통해 연락해 주세요.

## 9. 방침 변경

본 개인정보처리방침은 확장 프로그램의 기능 변경 또는 Chrome Web Store 정책 변경에 따라 업데이트될 수 있습니다. 변경 내용은 저장소 문서를 통해 공지합니다.
